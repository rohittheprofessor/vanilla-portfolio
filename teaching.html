Courses Taught

Theory of Computation (CS301)

Covers automata theory, formal languages, Turing machines, and computational complexity with emphasis on problem-solving and theoretical foundations.

Data Structures (CS202)

Introduces fundamental data structures including arrays, trees, graphs, and hashing with focus on algorithm efficiency and real-world applications.

Programming Fundamentals (CS101)

Builds strong programming foundations using C, Java, and Python, focusing on problem-solving, logic development, and structured programming.
